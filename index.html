選択中の音符

音符を選択すると、ここで音名を確認・修正できます。

コード / レッスン

「コード範囲」でドラッグ、または「小節送り」で、範囲内の音からコードを推定します。

使い方

  1. 画像を開く（⌘/Ctrl+V・ドロップも可）→ ✨自動認識
  2. 動画から取り込み：YouTube等を再生中のタブを共有→楽譜の範囲を1回囲む→以降は「次のページ」1クリックで取り込み（PCブラウザ）。
  3. 譜表設定：上段（ト音）の最下線→最上線をクリック。自動で下段（ヘ音）へ切替。段が複数あれば繰り返す。
  4. 音符：クリックで●を配置／選択。↑↓で半段移動。
  5. 消す：●クリックで1個、囲んでドラッグで範囲一括削除（拍子・テンポ・符尾の掃除に）。
  6. コード範囲：横ドラッグ → コード名。
⌘Z 取り消し（⌘⇧Z やり直し）/ sfn 臨時記号 / 0 解除 / Del 削除 / ●右クリックで即削除 / ←→ 小節送り
