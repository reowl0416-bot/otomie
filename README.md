# 🎼 オトミエ — 楽譜の音名・コードが見える

楽譜のスクリーンショットを読み込むと、**音名（C4, D4…）とコード（Am7, G7…）を自動で画像に重ねて表示**するWebアプリ。
音楽を教える人のための、対面／画面共有ライブ指導ツールです。

- **無料・登録不要・インストール不要**（ブラウザで開くだけ）
- **画像は端末内だけで処理** — 外部サーバーに一切送信されません
- 自動認識（五線・符頭・小節）＋ **必ず手で直せる**半自動設計
- **小節送りモード**：1小節ずつ音名・コード・音楽理論の解説を表示
- 自動保存・⌘Z取り消し・注釈つきPNG書き出し

## 使い方

1. **画像を開く**（⌘/Ctrl+V 貼り付け・ドラッグ&ドロップも可）→ **自動認識**
2. 誤り・抜けは手動修正（音符モードでクリック追加、消すモードで削除）
3. **小節送り開始** で1小節ずつ表示（←/→キー or ボタン）
4. **PNG** ボタンで注釈つき画像を書き出し

初めての方は「**サンプル楽譜で試す**」ボタンでそのまま体験できます。

## 対応環境

| 環境 | 対応 |
|---|---|
| PC（Chrome / Safari / Firefox） | ✅ フル対応 |
| iPad | ✅ フル対応（1本指=操作、2本指=移動・ズーム） |
| スマホ | △ 閲覧・小節送り中心（画面が小さいため編集は非推奨） |

## 対象の楽譜

きれいにレンダリングされた大譜表（ト音＋ヘ音）のスクリーンショット向けです。
**紙の楽譜の写真・手書き譜は対象外**です。自動認識はベストエフォートで、結果はすべて手動修正できます。

## プライバシーと著作権

- 画像・注釈データはブラウザ内（localStorage）にのみ保存されます。外部送信はありません。
- 市販楽譜等のスクリーンショットは、**私的利用・指導の正当な範囲**でご利用ください。

## 技術構成

- 単一HTML＋バニラJS＋Canvas。ビルド不要・オフライン動作・依存ゼロ
- `engine.js` — 音楽理論エンジン（音名・コード判定・解説。テスト38件）
- `cv.js` — 画像認識（五線・符頭・小節線検出。テスト18件）
- テスト実行: `node engine.test.js` / `node cv.test.js`（Node がなければ `cat engine.js engine.test.js | osascript -l JavaScript`）

## GitHub Pages で公開する手順

1. このフォルダを GitHub リポジトリに push（リポジトリ名の例: `otomie`）
2. リポジトリの **Settings → Pages → Source: Deploy from a branch → main / (root)** を選択
3. 数分後 `https://<アカウント名>.github.io/otomie/` で公開されます
4. **公開URLが決まったら** `index.html` の `<head>` 内にある `og:url` と `og:image` の
   `https://YOUR-ACCOUNT.github.io/otomie/` を実際のURLに書き換えて push（X のリンクカード用）
